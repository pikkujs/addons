# @pikku/addon-telegram

Telegram Bot API for messaging and chat management.

## Functions

**Messages:** `messageSend`, `messageSendPhoto`, `messageSendDocument`, `messageSendLocation`, `messageSendAnimation`, `messageSendAudio`, `messageSendVideo`, `messageSendSticker`, `messageSendMediaGroup`, `messageSendChatAction`, `messageEdit`, `messageDelete`, `messagePin`, `messageUnpin`
**Chat:** `chatGet`, `chatGetAdministrators`, `chatGetMember`, `chatLeave`, `chatSetDescription`, `chatSetTitle`
**Callbacks:** `callbackAnswerQuery`, `callbackAnswerInlineQuery`
**Files:** `fileGet`
**Triggers:** `onUpdate`

## Secrets

`TELEGRAM_BOT_TOKEN` — Telegram bot token from @BotFather (string)

## Dependencies

No additional runtime dependencies.
