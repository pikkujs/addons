# @pikku/addon-discord

Discord bot messaging and server management.

## Functions

**Messages:** `messageSend`, `messageGet`, `messageDelete`, `messageReact`
**Channels:** `channelGet`, `channelCreate`, `channelDelete`
**Members:** `memberList`, `memberRoleAdd`, `memberRoleRemove`

## Secrets

`DISCORD_BOT_TOKEN` — Discord bot token (string)

## Dependencies

No additional runtime dependencies.
