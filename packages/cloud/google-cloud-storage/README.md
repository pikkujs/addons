# @pikku/addon-google-cloud-storage

Google Cloud Storage bucket and object management.

## Functions

**Buckets:** `bucketCreate`, `bucketGet`, `bucketList`, `bucketUpdate`, `bucketDelete`
**Objects:** `objectUpload`, `objectGet`, `objectDownload`, `objectList`, `objectDelete`, `objectUpdateMetadata`

## Secrets

`GOOGLE_CLOUD_STORAGE_APP_CREDENTIALS` — Google OAuth2 credentials (OAuth2 flow)

## Dependencies

No additional runtime dependencies.
