# @pikku/addon-sendgrid

SendGrid email delivery and contact management.

## Functions

**Mail:** `mailSend`
**Lists:** `listCreate`, `listGet`, `listList`, `listUpdate`, `listDelete`
**Contacts:** `contactUpsert`, `contactGet`, `contactList`, `contactDelete`

## Secrets

`SENDGRID_API_KEY` — SendGrid API key (string)

## Dependencies

No additional runtime dependencies.
