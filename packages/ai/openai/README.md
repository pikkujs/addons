# @pikku/addon-openai

OpenAI API for chat, images, moderation, and embeddings.

## Functions

- `chatComplete` — generate chat completions
- `imageCreate` — generate images from prompts
- `textModerate` — moderate text for policy violations
- `textEmbedding` — generate text embeddings

## Secrets

`OPENAI_API_KEY` — OpenAI API key (string)

## Dependencies

- openai
