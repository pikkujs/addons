# @pikku/addon-whisper-asr

Speech-to-text via Whisper ASR webservice.

## Functions

- `transcribe` — transcribe audio to text
- `transcribeVerbose` — transcribe audio with timestamps and segments
- `translate` — translate audio to English text

## Secrets

No secrets required. Uses variable `WHISPER_ASR_BASE_URL` (defaults to `http://localhost:9000`).

## Dependencies

No additional runtime dependencies.
