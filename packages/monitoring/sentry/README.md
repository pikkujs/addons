# @pikku/addon-sentry

Sentry error tracking and organization management.

## Functions

**Issues:** `sentryIssueGet`, `sentryIssueGetMany`, `sentryIssueUpdate`, `sentryIssueDelete`
**Events:** `sentryEventGet`, `sentryEventGetMany`
**Organizations:** `sentryOrgGet`, `sentryOrgGetMany`, `sentryOrgCreate`, `sentryOrgUpdate`

## Secrets

`SENTRY_CREDENTIALS` — fields: token, baseUrl

## Dependencies

No additional runtime dependencies.
