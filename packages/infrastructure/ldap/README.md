# @pikku/addon-ldap

LDAP directory operations.

## Functions

- `ldapSearch` — search directory entries
- `ldapCreate` — create a directory entry
- `ldapUpdate` — update a directory entry
- `ldapDelete` — delete a directory entry
- `ldapCompare` — compare an attribute value

## Secrets

`LDAP_CREDENTIALS` — fields: url, bindDN, bindPassword, startTLS

## Dependencies

- ldapts
