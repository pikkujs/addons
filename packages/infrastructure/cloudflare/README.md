# @pikku/addon-cloudflare

Cloudflare zones and DNS record management.

## Functions

- `zonesList` — list zones
- `zonesGet` — get zone details
- `dnsRecordsList` — list DNS records for a zone
- `dnsRecordsCreate` — create a DNS record
- `dnsRecordsDelete` — delete a DNS record

## Secrets

`CLOUDFLARE_CREDENTIALS` — fields: apiKey

## Dependencies

No additional runtime dependencies.
