# @pikku/addon-ssh

SSH remote command execution and file transfer.

## Functions

- `sshExecute` — execute a command on a remote server
- `sshUpload` — upload a file to a remote server
- `sshDownload` — download a file from a remote server

## Secrets

`SSH_CREDENTIALS` — fields: host, port, username, password, privateKey, passphrase

## Dependencies

- ssh2
