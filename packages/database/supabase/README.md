# @pikku/addon-supabase

Supabase database operations.

## Functions

- `selectRows` — query rows from a table
- `insertRows` — insert rows into a table
- `updateRows` — update rows in a table
- `deleteRows` — delete rows from a table
- `upsertRows` — upsert rows into a table
- `rpc` — call a database function

## Secrets

`SUPABASE_CREDENTIALS` — fields: url, apiKey

## Dependencies

- @supabase/supabase-js
