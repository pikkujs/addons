# @pikku/addon-airtable

Airtable CRUD operations on records.

## Functions

- `listRecords` — list records in a table
- `getRecord` — get a single record
- `createRecord` — create a new record
- `updateRecord` — update an existing record
- `deleteRecord` — delete a record

## Secrets

`AIRTABLE_API_KEY` — Airtable Personal Access Token (string)

## Dependencies

No additional runtime dependencies.
