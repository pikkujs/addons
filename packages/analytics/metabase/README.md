# @pikku/addon-metabase

Metabase dashboard and question queries.

## Functions

- `questionsList` — list saved questions
- `questionsGet` — get a question and its results
- `dashboardsList` — list dashboards
- `dashboardsGet` — get a dashboard and its cards

## Secrets

`METABASE_CREDENTIALS` — fields: host, sessionToken

## Dependencies

No additional runtime dependencies.
