# @pikku/addon-google-analytics

Google Analytics 4 event tracking and reporting.

## Functions

- `eventSend` — send events via Measurement Protocol
- `eventValidate` — validate events before sending
- `reportRun` — run Data API reports
- `metadataGet` — get available report dimensions and metrics

## Secrets

`GOOGLE_ANALYTICS_CREDENTIALS` — fields: measurementId, apiSecret

## Dependencies

No additional runtime dependencies.
