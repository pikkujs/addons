# @pikku/addon-spreadsheet

Read and write CSV and XLSX spreadsheets.

## Functions

- `csvToJson` — parse CSV data to JSON
- `jsonToCsv` — convert JSON to CSV
- `xlsxToJson` — parse XLSX spreadsheet to JSON
- `jsonToXlsx` — convert JSON to XLSX spreadsheet

## Secrets

No secrets required.

## Dependencies

- csv-parse
- xlsx
