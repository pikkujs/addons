# @pikku/addon-compression

Compress and decompress data using gzip, deflate, and zip.

## Functions

- `gzipCompress` / `gzipDecompress` — gzip compression
- `deflateCompress` / `deflateDecompress` — deflate compression
- `zipCompress` / `zipDecompress` — zip archive compression

## Secrets

No secrets required.

## Dependencies

- fflate
