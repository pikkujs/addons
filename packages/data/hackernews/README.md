# @pikku/addon-hackernews

Hacker News API for stories, items, and users.

## Functions

- `hnGetItem` — get an item (story, comment, job, poll)
- `hnGetUser` — get a user profile
- `hnGetStories` — get top, new, best, ask, show, or job stories

## Secrets

No secrets required.

## Dependencies

No additional runtime dependencies.
