# @pikku/addon-edit-image

Image manipulation: resize, crop, rotate, blur, flip, composite, and convert.

## Functions

- `imageResize` — resize an image
- `imageCrop` — crop an image
- `imageRotate` — rotate an image
- `imageBlur` — apply blur to an image
- `imageFlip` — flip an image horizontally or vertically
- `imageComposite` — composite images together
- `imageConvert` — convert between image formats
- `imageMetadata` — extract image metadata

## Secrets

No secrets required.

## Dependencies

- sharp
