# @pikku/addon-qrcode

Generate and read QR codes.

## Functions

- `qrCodeGenerate` — generate a QR code as a buffer
- `qrCodeRead` — read and decode a QR code from an image
- `qrCodeToDataUrl` — generate a QR code as a data URL

## Secrets

No secrets required.

## Dependencies

- qrcode
- jsqr
- sharp
