# @pikku/addon-hubspot

HubSpot CRM for contacts, companies, deals, tickets, and engagements.

## Functions

**Contacts:** `contactUpsert`, `contactGet`, `contactList`, `contactDelete`, `contactSearch`
**Companies:** `companyCreate`, `companyGet`, `companyList`, `companyUpdate`, `companyDelete`, `companySearchByDomain`
**Deals:** `dealCreate`, `dealGet`, `dealList`, `dealUpdate`, `dealDelete`, `dealSearch`
**Tickets:** `ticketCreate`, `ticketGet`, `ticketList`, `ticketUpdate`, `ticketDelete`
**Engagements:** `engagementCreate`, `engagementGet`, `engagementList`, `engagementDelete`
**Contact Lists:** `contactListAdd`, `contactListRemove`
**Forms:** `formGetFields`, `formSubmit`

## Secrets

`HUBSPOT_API_KEY` — HubSpot Private App access token (string)

## Dependencies

No additional runtime dependencies.
